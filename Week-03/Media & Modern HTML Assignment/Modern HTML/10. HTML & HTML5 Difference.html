Difference Between HTML & HTML5

HTML HTML5
Major markup language used to display web pages on the Internet Major revision of HTML for describing the content and appearance of web pages
Supports vector graphics using other technologies or plugins Vector graphics are an integral part
Difficult to include and handle multimedia. Web sockets are not available Easy to include and handle multimedia. Web sockets are available and provide full duplex communication
Offline storage support is not good and does not support Geoloacation Offine storage support is good and supports Geoloacation
