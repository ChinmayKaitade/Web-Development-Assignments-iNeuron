Text Formatting in HTML

  1. Bold (b)
    • The Bold tag also supports the Event Attributes in HTML.
    • According to HTML5 specifications, the tag should be used as a LAST resort when no other tag is more appropraite.
    • Bold tag also supports the Global Attributes in HTML.
  2. underline (u)
    • The Underline tag represents some text that is unarticuated and styled differently from normal text, such as misspelled words or proper names in Chinese text.
    • It is used to underline the text enclosed within tag.
    • Underlined text is used to help draw attention to the text.
  3. Italic (i)
    • The Italic tag is often used to indicate a technical term, a phrase from another language, a though, a ship name, etc.
    • It defines a part of text in an alternative voice or mood.
    • Italic is a typeface or font style that slants to the right.
  4. Small
    • The Small tag defines smaller text(like copyright and other side-comments.)
    • The Small tag also supports the Global Attribute in HTML.
    • The Small tag also supports the Event Attribute in HTML.
  5. Emphasis (em)
    • The Emphasis tag is uded to define emphasized text.The content inside is typically displayed in Italic.
    • A Screen reader will pronounce the words in (em) with an emphasis, using verbal stress.
    • The (em) tag also supports Global Attribute in HTML & Event Attribute in HTML.
  6. Mark
    • The Mark tag defines the text should be marked or highlighted.
    • The Mark tag also supports the Global Attribute in HTML.
    • Thr Mark tag also supports the Event Attribute in HTML.
