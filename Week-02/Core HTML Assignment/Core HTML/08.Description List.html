Full Stack Web Development Tech Stack

HTML
HTML(HyperText Markup Language) is the code that is used to structure a web page and its content.HyperText Markup Language is the standard markup language for creating web pages and other online documents.
CSS
CSS makes Front-end of a website shine and it creates a great user Experience. Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in HTML or XML. It is used for styling purpose for webpage.
JavaScript
JavaScript allows developers to create dynamically updating content, use animations , pop-up menus, clickable buttons, control multimedia,etc. JavaScript is a programming language used to create interactive effects within web browsers.
Node.Js
Developers use Node.Js to create server-side Web applications, and it is perfect for data-intensive applications since it uses an asynchronous, event-driven model. Node.js is an open-source, cross-platform, JavaScript runtime environment that executes JavaScript code outside of a browser.
Express.Js
It is used to build a single page, multipage, and hybrid web applications. Express.js is a web application framework for Node.js designed for building web applications and APIs.
MongoDB
It is Used for developing scalable applications with evolving data schemas MongoDB is a NoSQL document-oriented database program that uses JSON-like documents with optional schemas.
React.Js
It is used for building interactive user interfaces and web applications quickly and efficiently with significantly less code than you would with Vanilla JavaScript React.js is a JavaScript library for building user interfaces or UI components.
